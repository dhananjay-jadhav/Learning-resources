## JavaScript Learning Resources

This repository contains a collection of concise, well-organized notes and summaries to help you brush up on JavaScript concepts quickly. Each topic is covered in a dedicated markdown file for easy reference.

### Topics Covered

- **JavaScript Fundamentals**  
	Key syntax, data types, and language basics.  
	_File: `nodejs/notes.md`_

- **Objects in JavaScript**  
	How to create, access, and manipulate object properties.  
	_File: `nodejs/object.md` & `nodejs/object-summary.md`_

- **Primitive vs. Reference Values**  
	Understanding stack vs. heap, copying behavior, and mutability.  
	_File: `nodejs/primitive-vs-reference-values.md` & `nodejs/primitive-vs-reference-values-summary.md`_

Feel free to explore each markdown file for detailed explanations and code examples. This collection is designed for quick review and self-improvement.
